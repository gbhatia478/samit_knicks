# Samit's Knicks Copium Terminal

An interactive, premium React/TypeScript/Vite web application built specifically to make fun of Samit's Knicks fandom and the NBA championship drought.

## 🚀 Features

1.  **iMessage Group Chat Dynamic**: Playable simulation of the actual group dynamic. Watch how Chris, Arvind, and Gaurav tease Samit, and how Samit responds with absolute confidence.
2.  **Knicks Championship Drought Counter**: Real-time counter showing the exact time since the Knicks last won a championship (May 10, 1973).
3.  **Excuse Generator**: Generates context-specific excuses from Samit on why the Knicks lost.
4.  **Trade Machine (Delusion Mode)**: Trade Mitchell Robinson and NYC Bagels for Nikola Jokic and see what NBA front offices think.
5.  **Knicks Ticket Refund Application**: A mock claim form that is legally rejected.

## 🛠️ Tech Stack

*   **Framework**: React + TypeScript + Vite
*   **Styling**: Pure CSS (using HSL variable-driven palettes)
*   **Design**: Glassmorphism cards, modern grid layouts, and custom micro-animations

## 💻 Running the App Locally

To start the development server, run the following commands in your terminal:

```bash
# Navigate to project directory
cd /Users/gauravbhatia/Desktop/samit-knicks-roast-app

# Start the dev server
npm run dev
```

The app will start at [http://localhost:5173/](http://localhost:5173/).
